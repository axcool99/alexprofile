# Alex Profile

Personal portfolio site for Alex Khoo built with React and Vite.  
The project focuses on cinematic motion, premium dark UI, and bilingual (CN/EN) content.

## Tech Stack
- React 18 + Vite
- GSAP + ScrollTrigger
- three.js + @react-three/fiber + @react-three/drei
- Iconify (line-md)
- Custom CSS (design-driven styling)

## Local Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
- All assets are local (no external CDN embeds).
- Content is stored in `content.json`.
- Custom icons live in `src/components/LineIcon.jsx`.
- Animations are tuned separately for desktop and mobile.
